# Weekly Summary Generator

Read my context from `context/about-me.md`, then read ALL daily entries for the current week
from the `daily/` folder. Also check `reviews/` for last week's review file — use it for
trend comparison.

## Your Job

Generate a draft weekly review using the template from `templates/weekly-review.md`.
Fill it in based on what you find in the daily entries.

### 1. Wins
Pull out anything I did well this week — even small things. I need to see progress.

### 2. Learning Summary
What topics came up? Going deep on anything or just skimming? Cross-reference `context/learning-topics.md`.

### 3. Health / Fitness
How many days did I log movement? Specific. "3 out of 7 days" not "you did okay."
Same for sleep and reading.

### 4. Honest Assessment
Where did I fall short? Blank entries, missed days, categories consistently skipped?
Don't sugarcoat.

### 5. Recurring Tags & Themes
- Count how often each frontmatter tag appeared this week
- Identify any theme from prose that kept coming up but isn't tagged yet
- Flag: Emerging (new this week), Recurring (ongoing), Fading (was there, now quiet)

### 6. Trend vs. Last Week
If last week's review exists in `reviews/`, compare:
- Mood average: up or down?
- Exercise frequency: more or fewer days?
- Learning: maintained or dropped?
- Any metric that moved notably — call it out with numbers

### 7. Next Week Focus
Based on everything: 1-2 specific priorities for next week.
Tie to `goals/2026-goals.md` if relevant.

## Output
Write the full weekly review to `reviews/YYYY-WXX.md` using the template format.
Be direct and specific, not generic.
