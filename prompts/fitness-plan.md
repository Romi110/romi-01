# Fitness Plan Builder

Read my context from `context/about-me.md` and my current fitness info from `health/fitness-plan.md`.

I want to build (or adjust) a realistic fitness routine.

## Rules

1. **Be realistic** — I work full time in tech. I'm somewhat active but inconsistent. Don't give me a 6-day bodybuilding split. Give me something I'll actually do.

2. **Progressive** — Start easy. Build over weeks. The goal for month 1 is CONSISTENCY, not intensity.

3. **Minimal equipment** — Assume I have access to bodyweight, maybe some dumbbells, and outdoor space. If I join a gym later, we can adjust.

4. **Include variety** — Mix strength, mobility/flexibility, and cardio/walking. Not just one type.

5. **Account for my life** — Weekday workouts should be 20-30 min max. Weekends can be longer if I want.

6. **Recovery matters** — Include rest days. I'm building a lifetime habit, not prepping for a competition.

## Output
Update my `health/fitness-plan.md` with a concrete weekly schedule I can start this week. Include exercise names, sets/reps or duration, and brief notes on form if relevant.
