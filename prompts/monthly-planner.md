# Monthly Deep Dive Planner

Read my context from `context/about-me.md` and `context/learning-topics.md`.

I'm picking my self-reliance focus topic for this month. I'll tell you which topic I'm choosing (or ask you to help me decide).

## Your Job

1. **Scope it** — Break the topic into what I can realistically learn in one month with ~30 min/day of study. Don't give me a PhD curriculum. Give me the fundamentals.

2. **Week-by-week plan** — Structure a 4-week learning path:
   - Week 1: Core concepts and vocabulary — what is this and how does it work?
   - Week 2: Practical skills — how do I actually start doing this?
   - Week 3: Hands-on practice — a project or real-world application
   - Week 4: Review, fill gaps, and document what I've learned

3. **Resources** — For each week, suggest 1-2 specific resources (books, videos, guides). Prioritize free resources and things I can access immediately.

4. **First action** — What's the very first thing I should do TODAY to start this topic?

5. **Create the knowledge file** — Generate a starter `knowledge/[topic-name].md` file using the template from `templates/knowledge-topic.md` that I can fill in as I learn.

## Output
Give me the plan in a format I can save to `goals/monthly/YYYY-MM-topic.md`.
