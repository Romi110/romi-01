# romi-01

A personal system for learning, self-reliance, health, and daily growth.

## Structure

```
romi-01/
├── context/          # Who I am, my values, life context (feed this to Claude)
├── daily/            # Daily log entries organized by month
│   └── 2026-03/      # One file per day
├── knowledge/        # Deep-dive notes on topics I'm learning
├── reading/          # Book and article log
├── health/           # Fitness plans, tracking, body notes
├── goals/            # Goal definitions and progress
├── reviews/          # Weekly and monthly reflections
└── templates/        # Reusable templates for entries
```

## How to Use This

**Primary workflow: VS Code + Claude extension**
See `knowledge/vscode-claude-workflow.md` for the full guide.

Quick version:
1. Open this repo in VS Code
2. Reference `@context/about-me.md` when starting Claude conversations
3. Create daily entries from the template, ask Claude questions, take notes
4. Commit regularly with Git

## Philosophy

- Short entries beat no entries. Five lines on a busy day is a win.
- Learn something every day, even if it's small.
- Self-reliance is a spectrum, not a destination. Every skill learned is progress.
- This system serves you. Change it whenever it stops working.
