# Template Note

This template is now auto-generated by `scripts/new-day.js` from `config/categories.json`.

To create a new daily entry, run:
```
node scripts/new-day.js
```

The script will:
1. Read all categories from the config
2. Generate YAML frontmatter for trackable categories
3. Create a section for each category with its prompt
4. Save to `daily/YYYY-MM/YYYY-MM-DD.md`

To add or change categories, edit `config/categories.json` and the next entry you create will reflect the changes.
